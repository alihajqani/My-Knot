# Dependency Injection (DI)

## ğŸ“Œ The Concept
**Don't build your tools inside; ask for them from the outside.**

*   **Bad Way (Hardcoded):** A flashlight with a battery soldered inside. If the battery dies, you throw away the flashlight.
*   **Good Way (DI):** A flashlight with a slot. You can "inject" any brand of battery (Sony, Duracell, or a Fake Testing Battery).

### TypeScript (The Good Way - Dependency Injection)

```TypeScript
interface Engine {
    start(): void;
}

class Car {
    private engine: Engine;

    // âœ… Good: We ASK for the engine.
    // We can pass a RealEngine or a FakeTestEngine here.
    constructor(engine: Engine) {
        this.engine = engine;
    }

    drive() {
        this.engine.start();
    }
}
```
